# translate-xlsx

## 安装

1、环境准备

```json
"engines": {
    "node": ">=16.18.0",
    "pnpm": ">=7.23.0"
  },
```

### nvm 安装

https://github.com/nvm-sh/nvm

### pnpm 安装
```sh
npm i -g pnpm
```

### install

```sh
pnpm install
```


## 用法

1、根目录放入需要翻译的脚本，命名为`translate.xlsx`

## 启动

```sh
pnpm start 
```
